### 题目
计算字符串中没有重复字符的最长子串

### 思路


### 代码

```
int lengthOfLongestSubstring(char * s){
    int left = 0;
    int right = 0;
    bool hash[256] = {0};
    int res = 0;
    while (right < strlen(s) && hash[s[right]] == 0) {
            hash[s[right]] = true;
            right++;
    }
    res = right - left;

    while (left < strlen(s)) {
        while (right < strlen(s) && hash[s[right]] == 0) {
            hash[s[right]] = true;
            right++;
        }
        res = res > (right - left) ? res : (right - left);
        hash[s[left]] = false;
        left++;
    }
    return res;
}
```
