### fib数列

使用cash来减少重复计算
掌握： 递归 + cash 的思路

### 代码
```
const int MOD = 1e9 + 7;
int memo[101] = {0,1};
int fib(int n){
    if (n <= 1) {
        return n;
    }
    if (memo[n] != 0 && memo[n] != 1) {
        return memo[n];
    }
    memo[n] = fib(n - 1) % MOD + fib(n - 2) % MOD;
    return (memo[n]) % MOD;
}
```
