```
int sum = 0;
for (int i = 0; i < 3; i++) {
  sum += sum * 10 + a[0];
}
```

2、计算除法中的小数部分。

```
int a;
int b; // 2个数字

int total = a / b; //先得到整数部分

int fator = a % b;
while (fator) { // 直到除尽。
  int fator1 = (fator * 10) / b; // 先把小数部分扩大10倍，在对b取整，把得到的值赋值给 fator1,这就是小数部分的第一位。
  fator = fator * 10 % b; // 把小数部分扩大10倍后，取余，就是第二位小数的被除数。
}
```
