
回溯其实就是从一开始不断前进，直到遇到阻碍行不通或者是走到结束了，就向回走一步，在继续向下探索，如果这也不能继续探索，则继续回退，依次为开始再次向下搜索。不断重复这个过程。

回溯的过程中需要考虑的问题： 1、路径，即已经做出的选择； 2、选择列表，即当前可以做的选择；3、结束条件，即无法再做选择的条件。

回溯的框架代码如下：

```
result [] = {};
void backtrace(路径， 选择列表)
{
  if (满足条件) {
    把当前路径添加到结果
    return；
  }
  for(选择 in  选择列表) {
    做选择
    backtrace(路径， 选择列表)；
    撤销选择
  }
}
```
